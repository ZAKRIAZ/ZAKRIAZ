### 🕯️ The Craft

I'm **Zakariae**, an integration & full-stack developer from **Rabat, Morocco**.

By day I bind business systems together — **NetSuite · Celigo · Workato · EDI · REST APIs** — so data flows without a soul touching it. By night I raise apps, scrapers, and **AI agents on the Model Context Protocol (MCP)** from whatever idea won't let me rest.

My favorite creation is the **[AI Universal Scraper](https://github.com/ZAKRIAZ/ai-universal-scraper)** — point an LLM at any page and it drags back clean, structured data from the void.

> *"I conjure order from chaos — and automate the rest."*
